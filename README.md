# lab6
Repositorio del codigo del laboratorio 6 (opcional) de Plataformas abiertas

Este codigo usa un arreglo de 5 lineas con 3 valores: nombre, edad y altura (cm). Los ordena usando qsort de tal manera que los nombres estan alfabeticamente, y la edad y altura se ordenan de menor a mayor

Para compilar:
	-Asegurate estar dentro de la carpeta lab6 y haz el comando "make"

Esto crea un archivo ejecutable llamado "Person"

Para ejecutar:
	-Usa el comando "./Person"

Para limpiar:
	-Usa "make clean" y elimina los archivos .o y el ejecutable

Para valgrind:
	-Usa "make valgrind"
